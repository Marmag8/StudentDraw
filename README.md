# StudentDraw

## Overview
StudentDraw is a .NET MAUI application designed to provide an interactive drawing experience for students. The application allows users to create, save, and share their artwork in a user-friendly environment.

## Features
- Interactive drawing tools
- Save and share artwork
- User-friendly interface

## Styling
For this project, we prioritize the use of CSS for styling within the .NET MAUI framework. This approach allows for a more flexible and maintainable design, enabling developers to easily customize the look and feel of the application.

## Getting Started
To get started with StudentDraw, follow these steps:

1. Clone the repository:
   git clone https://github.com/yourusername/StudentDraw.git

2. Navigate to the project directory:
   cd StudentDraw

3. Install the necessary dependencies:
   dotnet restore

4. Run the application:
   dotnet run
